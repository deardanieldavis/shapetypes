Shapetypes

A 2d geometry library based loosely on the Open NURBS API.

Features
  • Essential geometry types – point, line, vector, rectangle, polyline, and polygon
  • Basic transformations – rotate, translate, and plane to plane
  • Intersections, booleans, and containment
  • Lightweight and written in Typescript
  • Lots of documentation

Getting started

npm install shapetypes

import {Point, Plane, Rectangle} from 'shapetypes';
let rectangle = new Rectangle(Plane.worldXY, 10, 20);
console.log(rectangle.area);
//>200
console.log(rectangle.contains(new Point(5, 5)));
//>true

Alternatives
  • Rhino3dm: A web assembly port of Rhino Common. This is a much more extensive library that supports 3d geometry.
  • VerbNurbs: A cross-platform Nurbs library.
  • Javascript Topology Suite: A library of geometric operations primarily for GIS data.
  • Turf JS: Another library of geometric operations primarily for GIS data.
